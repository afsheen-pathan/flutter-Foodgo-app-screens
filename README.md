# 🍔 Foodgo – Flutter Food Delivery App

Foodgo is a modern and visually appealing **Food Delivery Mobile Application** built using Flutter.  
It demonstrates a complete food ordering flow with multiple screens and interactive UI components.

---

## 📱 Screens Included

- Splash Screen  
- Home Screen  
- Product Detail Screen  
- Payment Screen  
- Customize Order Screen  
- Chat Support Screen  
- Profile Screen  
- Success Screen  

---

## ✨ Features

- Clean and modern UI design  
- Category-based food browsing  
- Search and filter interface  
- Favorite items functionality  
- Spicy level and portion control  
- Order summary and payment UI  
- Food customization (toppings & sides)  
- Chat interface design  
- User profile management  
- Smooth navigation between screens  

---

## 🛠️ Tech Stack

- Flutter  
- Dart  
- Material UI
